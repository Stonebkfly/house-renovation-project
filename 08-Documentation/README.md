# Documentation Folder

## Purpose
This folder stores all important project documentation including photos, receipts, and contracts.

## Folder Structure

### Before-Photos
Store all photos taken before renovation begins:
- Exterior photos (all angles)
- Each room from multiple angles
- Close-ups of problem areas
- Existing fixtures and features
- Date stamp all photos
- Recommended: Use descriptive filenames (e.g., "Kitchen-North-Wall-Before.jpg")

### After-Photos
Store all photos taken after renovation completes:
- Same angles as before photos for comparison
- Close-ups of new features
- Detail shots of finishes
- Final exterior views
- Date stamp all photos
- Recommended: Mirror the naming from before photos (e.g., "Kitchen-North-Wall-After.jpg")

### Receipts
Store all purchase receipts:
- Organize by vendor or by date
- Scan physical receipts (they fade over time)
- Include both materials and labor receipts
- Keep warranty receipts separate in warranties folder
- Document expense in budget tracker when filing receipt

### Contracts
Store all signed contracts and agreements:
- General contractor agreement
- Subcontractor agreements
- Material supplier contracts
- Design/architecture agreements
- Signed change orders
- Lien waivers
- Final completion certificates

## Best Practices

### Photo Documentation
1. **Before Starting:**
   - Photograph every room from all angles
   - Take photos of any existing damage
   - Document measurements with a tape measure in photo
   - Photograph neighboring properties if exterior work

2. **During Construction:**
   - Weekly progress photos (store in 09-Progress-Photos)
   - Before covering up: electrical, plumbing, framing
   - Any issues or damage discovered
   - Delivery of major materials

3. **After Completion:**
   - Match angles from before photos
   - Detail shots of craftsmanship
   - All new features and fixtures
   - Final walkthrough photos

### Receipt Management
- File receipts immediately upon purchase
- Note what was purchased on receipt if not clear
- Cross-reference with budget tracker
- Scan or photograph all receipts as backup
- Keep receipts for tax purposes (may be deductible)

### Contract Management
- Never start work without signed contract
- Review all contracts carefully before signing
- Ensure payment terms are clear
- Verify insurance and license info
- Keep original contracts in safe place
- Store copies here for easy reference

### For Home Sale or Appraisal
Having this documentation increases home value and makes selling easier:
- Before/after photos show improvements
- Receipts document investment
- Permits show work was done legally
- Warranties transfer to new owners
- Contracts prove professional installation

## Tips
- Set up automatic backup of photo folders
- Use consistent naming conventions
- Date stamp everything
- Create digital copies of paper documents
- Review and organize weekly during project
- Keep a master list of all major purchases
